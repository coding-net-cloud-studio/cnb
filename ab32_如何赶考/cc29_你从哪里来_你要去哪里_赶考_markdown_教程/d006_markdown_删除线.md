<font size="4">Markdown 删除线</font>

用俩 `~` 左右包起来即可，也可以用HTML的 `<s> </s>` 标签来实现（不推荐）。

<font color="#FF7F00">语法示例：</font>

```
~~Markdown 删除线内容~~

<s>HTML 删除线内容</s>
```

<font color="#7093DB">渲染效果：</font>

~~Markdown 删除线内容~~

<s>HTML 删除线内容</s>
