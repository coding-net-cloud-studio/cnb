<font size="4">Markdown 特殊字符</font>

<font color="#FF7F00">语法示例：</font>

```
空格符：&nbsp;
小于号：&lt;
大于号：&gt;
和号：&amp;
人民币：&yen;
版权：&copy;
注册商标：&reg;
摄氏度：&deg;
正负号：&plusmn;
乘号：&times;
除号：&divide;
平方（上标²）：&sup2;
立方（上标³）：&sup3;
```

<font color="#7093DB">渲染效果：</font>

空格符：&nbsp;
小于号：&lt;
大于号：&gt;
和号：&amp;
人民币：&yen;
版权：&copy;
注册商标：&reg;
摄氏度：&deg;
正负号：&plusmn;
乘号：&times;
除号：&divide;
平方（上标²）：&sup2;
立方（上标³）：&sup3;

注：在空格符的上下方各插入一个空行，即可在渲染时显示空白行。
